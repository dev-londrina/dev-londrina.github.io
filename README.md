## Dependencias

* Ruby 2.1.2 - Você pode usar o [RVM](http://rvm.io)

## Setup

1. Instalar as dependências acima
2. Clonar o projeto
  `git clone git@github.com:dev-londrina/dev-londrina.github.io.git`
3. Entrar no projeto
  `cd dev-londrina`
4. Instalar as dependências
  `bundle install`
5. Rodar o projeto
  `jekyll serve -w`
6. Acessar http://localhost:4000
